## Docs
  * "/": API docs
  * "/signup": signup
  * "/signin": signin
  * "/signout": signout
  * "/users": get all users
  * "/user/:userId": get/update/delete user
  * "/posts": get all posts
  * "/post/new/:userId": create new post
  * "/posts/by/:userId": get posts by user
  * "/post/:postId": update/delete post
